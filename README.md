# kindlePush

kindle 图书推送

- [x] server:
mqtt服务器，作为消息传输的中介

- [ ] file
文件服务器，存储文件，并通过mqtt与web服务器交换文件

邮件服务器，提供实际邮件发送（已经抽象了一个sendMail函数了）
